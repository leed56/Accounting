# BizManager / Smart Business Suite

Premium business management for Sri Lankan small businesses — income, expenses, staff, payroll, approvals, and AI insights.

## Quick Start

```bash
pnpm install
pnpm dev:web      # http://localhost:3000
pnpm dev:mobile   # Expo Go
```

Demo mode works without Supabase (Royal Travels Office sample data).

## Stack

- **Web:** Next.js 15, Tailwind, TanStack Query
- **Mobile:** Expo 52, Expo Router
- **Backend:** Supabase (PostgreSQL, Auth, RLS)
- **Languages:** English, Sinhala, Tamil

See [docs/setup.md](docs/setup.md) for full setup.
